# Security Policy

## Supported Project

This repository contains the source code for the PP Studio website.

Security updates are applied to the current production version of the website only.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by email:

info@ppstudio.cz

Please include:
- a short description of the issue,
- steps to reproduce it,
- affected URL or file, if applicable,
- your contact information.

We will review the report as soon as possible and take appropriate action.
